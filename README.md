# 🏥 Clinic Management JavaFX

## Description
Application de gestion de clinique développée en JavaFX selon l'architecture MVC.

## Fonctionnalités
- Gestion des patients
- Gestion des médecins
- Gestion des rendez-vous
- Tableau de bord
- Recherche
- Modification
- Suppression avec confirmation

## Technologies
- Java 25
- JavaFX
- JDBC
- MySQL
- CSS
- MVC
